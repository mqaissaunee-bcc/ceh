🔧 Enumeration Fundamentals

What is Enumeration?

Enumeration is the process of extracting detailed information about network resources, users, groups, shares, services, and system configurations from target systems. It builds upon the information gathered during scanning to create a comprehensive picture of the target environment.

1. Target Identification

Identify live systems and accessible services from previous scanning phases

2. Service Enumeration

Extract detailed information from discovered services and applications

3. User & Group Enumeration

Discover user accounts, groups, and organizational structure

4. Share & Resource Enumeration

Identify network shares, databases, and accessible resources

5. Information Compilation

Organize and analyze gathered information for attack planning

🔍 Enumeration vs. Scanning

🔍 Scanning

  • Identifies live systems and open ports
  • Basic service detection
  • Network topology discovery
  • Less intrusive and noisy
  • Broader scope, less detail

📋 Enumeration

  • Extracts detailed system information
  • User accounts and groups
  • Network shares and resources
  • More intrusive and detectable
  • Narrower scope, deep detail

🌐 NetBIOS Enumeration

Windows network service discovery

🌐 NetBIOS & SMB Enumeration

🔍 Understanding NetBIOS and SMB

NetBIOS (Network Basic Input/Output System) and SMB (Server Message Block) are fundamental Windows networking protocols that provide rich information about systems, users, and network resources. These protocols are goldmines for enumeration activities.

🎯 NetBIOS Enumeration Techniques

📋 NetBIOS Name Table

nbtstat -A target_ip

  • Computer name and domain
  • Workgroup information
  • User currently logged on
  • MAC address discovery

🔍 NBTScan Discovery

nbtscan -r 192.168.1.0/24

  • Network-wide NetBIOS scanning
  • Bulk system identification
  • User enumeration across network
  • Share discovery automation

📁 SMB Enumeration with enum4linux

enum4linux is a comprehensive tool for enumerating information from Windows and Samba systems through SMB. It automates the process of gathering users, shares, groups, and system information.

🗂️ Share Enumeration

enum4linux -S target

  • Available network shares
  • Share permissions and access
  • Hidden and administrative shares
  • Share contents discovery

👥 User Enumeration

enum4linux -U target

  • Domain and local users
  • User account details
  • Password policy information
  • User group memberships

👥 Group Enumeration

enum4linux -G target

  • Domain and local groups
  • Group membership information
  • Administrative groups
  • Group permissions and rights

🔒 Policy Information

enum4linux -P target

  • Password policy settings
  • Account lockout policies
  • Domain policy information
  • Security settings

🛡️ SMB Null Session Attacks

Null sessions exploit a vulnerability in older Windows systems where anonymous connections can be established to gather information without authentication.

⚠️ Null Session Commands:

  • net use \\target\ipc$ "" /user:"" - Establish null session
  • net view \\target - View available shares
  • net user /domain - Enumerate domain users
  • net group /domain - Enumerate domain groups

📊 SNMP Enumeration

🔍 Understanding SNMP

Simple Network Management Protocol (SNMP) is used for network management and monitoring. SNMP agents on network devices provide a wealth of information through Management Information Base (MIB) objects, making it a prime target for enumeration.

🎯 SNMP Architecture

🏢 SNMP Manager

  • Queries SNMP agents for information
  • Receives traps and notifications
  • Manages network monitoring
  • Centralized management system

🤖 SNMP Agent

  • Runs on managed devices
  • Responds to manager queries
  • Sends traps when events occur
  • Maintains MIB information

📊 MIB (Management Information Base)

  • Hierarchical database of objects
  • Object Identifiers (OIDs)
  • System and device information
  • Standardized data structure

🔑 Community Strings

  • Authentication mechanism
  • Read-only: "public" (default)
  • Read-write: "private" (default)
  • Custom community strings

🔍 SNMPwalk Information Extraction

SNMPwalk is used to retrieve all available information from a device's SNMP agent by walking through the MIB tree.

🖥️ System Information

snmpwalk -v2c -c public target 1.3.6.1.2.1.1

  • System description and uptime
  • Hardware and software information
  • Contact and location details
  • Device capabilities

🔌 Network Interfaces

snmpwalk -v2c -c public target 1.3.6.1.2.1.2

  • Interface names and descriptions
  • IP addresses and configurations
  • Traffic statistics
  • Interface status and speed

⚙️ Running Processes

snmpwalk -v2c -c public target 1.3.6.1.2.1.25.4

  • Process names and IDs
  • Running applications
  • Process parameters
  • Resource utilization

👤 User Accounts

snmpwalk -v2c -c public target 1.3.6.1.4.1.77.1.2.25

  • Local user accounts
  • User session information
  • Login statistics
  • Account properties
