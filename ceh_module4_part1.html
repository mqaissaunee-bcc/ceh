üîß Enumeration Fundamentals

What is Enumeration?

Enumeration is the third phase of ethical hacking, following reconnaissance and scanning. It involves extracting detailed information about identified services, users, shares, and network resources.

Key Objectives:

  • Extract usernames, machine names, network resources
  • Collect information about network shares
  • Identify system vulnerabilities and misconfigurations
  • Gather application and service banners

Enumeration vs Scanning

Scanning Enumeration
Identifies live hosts and open ports Extracts detailed information from services
Non-intrusive, passive approach More intrusive, active information gathering
Broader network mapping Deep dive into specific services

üåê NetBIOS & SMB Enumeration

NetBIOS (Network Basic Input/Output System)

NetBIOS operates on ports 137-139 and provides name resolution and session services for Windows networks.

Key NetBIOS Commands:

nbtstat -A <IP> # Remote machine name table
nbtstat -a <hostname> # Name table by hostname
net view \\<IP> # Available shares
net use \\<IP>\IPC$ "" /u:"" # Null session connection

SMB (Server Message Block) Enumeration

SMB runs on port 445 and is used for file/printer sharing in Windows environments.

SMB Enumeration Tools:

  • enum4linux: Comprehensive SMB enumeration script
  • smbclient: Connect to SMB shares
  • rpcclient: Execute RPC commands
  • smbmap: SMB share enumeration and interaction
enum4linux -a <IP>
smbclient -L //<IP> -N
smbmap -H <IP> -u anonymous

üìä SNMP Enumeration

Simple Network Management Protocol (SNMP)

SNMP operates on UDP port 161 and is used for network device management. It's often misconfigured with default community strings.

Common Community Strings:

  • public (read-only - default)
  • private (read-write)
  • manager
  • admin
  • cisco

SNMP Enumeration Techniques

snmpwalk -v1 -c public <IP> 1.3.6.1.4.1
snmpget -v1 -c public <IP> 1.3.6.1.2.1.1.1.0
onesixtyone -c community.txt <IP>
nmap --script snmp-* <IP>

Valuable SNMP OIDs:

  • 1.3.6.1.2.1.1.1.0: System description
  • 1.3.6.1.2.1.1.5.0: Hostname
  • 1.3.6.1.2.1.4.20.1.1: IP addresses
  • 1.3.6.1.2.1.25.1.6.0: System processes
  • 1.3.6.1.2.1.25.4.2.1.2: Running programs
