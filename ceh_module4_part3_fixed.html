‚òÅÔ∏è Cloud Infrastructure Enumeration

AWS (Amazon Web Services) Enumeration

AWS is the largest cloud provider with hundreds of services. Proper enumeration can reveal S3 buckets, EC2 instances, and IAM configurations.

AWS CLI Enumeration Commands:

aws s3 ls
aws ec2 describe-instances
aws iam list-users
aws rds describe-db-instances
aws lambda list-functions

S3 Bucket Enumeration Tools:

  • AWS CLI: Official command-line interface
  • S3Scanner: Find open S3 buckets
  • Bucket Finder: Discover S3 bucket names
  • AWSBucketDump: Download S3 bucket contents

Azure Enumeration

Microsoft Azure provides extensive cloud services. Enumeration focuses on Active Directory, storage accounts, and virtual machines.

az account list
az vm list
az storage account list
az ad user list
az keyvault list

Google Cloud Platform (GCP) Enumeration

GCP enumeration targets Compute Engine instances, Cloud Storage buckets, and IAM policies.

gcloud compute instances list
gcloud storage buckets list
gcloud projects list
gcloud iam service-accounts list

üóÑÔ∏è Database Service Enumeration

MySQL Enumeration (Port 3306)

MySQL is one of the most popular database systems. Enumeration can reveal database names, tables, and user accounts.

mysql -u root -p -h <IP>
nmap --script mysql-* <IP>
mysql -u root -h <IP> -e "show databases;"
sqlmap -u "http://target/page.php?id=1" --dbs

PostgreSQL Enumeration (Port 5432)

PostgreSQL enumeration focuses on extracting database schemas, user roles, and configuration information.

psql -U postgres -h <IP>
nmap --script pgsql-* <IP>
\l (list databases)
\du (list users)

Microsoft SQL Server (Port 1433)

MSSQL enumeration can reveal database instances, authentication methods, and linked servers.

sqlcmd -S <IP> -U sa
nmap --script ms-sql-* <IP>
enum4linux -a <IP>
sqsh -S <IP> -U sa

MongoDB Enumeration (Port 27017)

MongoDB is a NoSQL database that often lacks proper authentication. Enumeration can reveal collections and documents.

mongo <IP>:27017
show dbs
use <database>
show collections
db.collection.find()

Database Enumeration Tools

  • SQLMap: Automated SQL injection and database takeover
  • NoSQLMap: NoSQL database exploitation framework
  • DBPwAudit: Database password auditing tool
  • OraclePasswordcracker: Oracle database password cracking
  • Redis-cli: Redis database enumeration

Directory and File Enumeration

Discovering hidden directories, files, and backup copies is crucial for web application enumeration.

Directory Bruteforce Tools:

dirb http://<target> /usr/share/dirb/wordlists/common.txt

Technology Stack Identification

Identifying the underlying technologies helps focus enumeration efforts on specific vulnerabilities.

whatweb <target>
wappalyzer (browser extension)
nmap --script http-enum <target>
nikto -h <target>

Common Technology Indicators:

  • PHP: .php extensions, X-Powered-By headers
  • ASP.NET: .aspx extensions, __VIEWSTATE parameters
  • Java: .jsp extensions, JSESSIONID cookies
  • Node.js: Express server headers
  • Python: Django/Flask frameworks

API Enumeration

Modern applications heavily rely on APIs. Proper enumeration can reveal endpoints, parameters, and authentication mechanisms.

API Discovery Techniques:

  • Check /api/, /v1/, /v2/ endpoints
  • Look for Swagger/OpenAPI documentation
  • Analyze JavaScript files for API calls
  • Use tools like Postman for API exploration
  • Check for GraphQL endpoints (/graphql, /graphiql)
curl -X GET http://<target>/api/
curl -X OPTIONS http://<target>/api/
ffuf -w api-wordlist.txt -u http://<target>/FUZZ
arjun -u http://<target>/api/endpoint
