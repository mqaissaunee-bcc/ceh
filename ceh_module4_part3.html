☁️ Cloud Enumeration Techniques

🌩️ Modern Cloud Attack Surface

Cloud enumeration represents a paradigm shift from traditional network reconnaissance. Instead of scanning IP ranges, attackers now focus on discovering cloud resources, understanding identity and access management (IAM) configurations, and exploiting cloud-specific vulnerabilities across multiple service providers.

🟠 Amazon Web Services (AWS)

aws s3 ls --no-sign-request

  • S3 bucket enumeration and analysis
  • EC2 instance metadata extraction
  • IAM role and policy enumeration
  • Lambda function discovery
  • CloudFormation stack analysis

🔵 Microsoft Azure

az storage account list

  • Storage account enumeration
  • Azure AD tenant discovery
  • Virtual machine reconnaissance
  • Key Vault secret enumeration
  • Function App analysis

🔵 Google Cloud Platform

gsutil ls gs://bucket-name

  • Cloud Storage bucket discovery
  • Compute Engine enumeration
  • IAM policy analysis
  • Cloud Functions reconnaissance
  • BigQuery dataset discovery

Impact: Data exposure, compliance violations

🔑 Overprivileged IAM

Impact: Privilege escalation, lateral movement

🌐 Exposed Services

Impact: Unauthorized access, data manipulation

Impact: Credential theft, service enumeration

🗄️ Database Service Enumeration

🔍 Modern Database Landscape

Database enumeration has evolved far beyond traditional SQL servers. Today's applications use diverse database technologies including NoSQL databases, cloud-native solutions, and distributed systems. Each requires specialized enumeration techniques and presents unique attack vectors.

🗃️ Traditional SQL Databases

nmap -sV --script mysql-enum target

  • MySQL, PostgreSQL, MSSQL enumeration
  • Version detection and vulnerability analysis
  • User account and privilege enumeration
  • Database schema and table discovery

📊 NoSQL Databases

mongo --host target --eval "db.version()"

  • MongoDB, CouchDB, Cassandra
  • Document and collection enumeration
  • Authentication bypass techniques
  • Data extraction and analysis

☁️ Cloud Database Services

aws rds describe-db-instances

  • AWS RDS, Azure SQL, Cloud SQL
  • DynamoDB and CosmosDB enumeration
  • Configuration and access analysis
  • Backup and snapshot discovery

⚡ In-Memory Databases

redis-cli -h target info

  • Redis and Memcached enumeration
  • Key-value pair extraction
  • Configuration and security analysis
  • Session and cache data discovery

🔍 Search and Analytics

curl -X GET "target:9200/_cat/indices"

  • Elasticsearch and Solr enumeration
  • Index and document discovery
  • Query injection techniques
  • Cluster configuration analysis

🕸️ Graph Databases

cypher-shell -a bolt://target

  • Neo4j and ArangoDB enumeration
  • Relationship and node discovery
  • Cypher query injection
  • Graph structure analysis

🎯 Database Attack Vectors

🚨 Common Database Vulnerabilities:

  • Authentication Bypass: Default credentials, weak authentication mechanisms
  • Injection Attacks: SQL injection, NoSQL injection, LDAP injection
  • Privilege Escalation: Database function abuse, stored procedure exploitation
  • Data Exposure: Unencrypted data, backup file access, log file analysis
  • Configuration Issues: Open network access, excessive privileges, weak encryption

🌐 Web Application Enumeration

🚀 Modern Web Application Architecture

Modern web applications are complex ecosystems involving single-page applications (SPAs), microservices, APIs, content delivery networks (CDNs), and serverless functions. Effective enumeration requires understanding these architectures and the unique attack surfaces they present.

🔍 Application Discovery

gobuster dir -u target -w wordlist

  • Directory and file enumeration
  • Hidden endpoint discovery
  • Backup and config file detection
  • Version control system exposure

🛠️ Technology Stack Analysis

whatweb target

  • Framework and library identification
  • Server and language detection
  • Third-party component analysis
  • CDN and hosting provider discovery

🔌 API and Microservice Enumeration

ffuf -w api-wordlist -u target/FUZZ

  • REST API endpoint discovery
  • GraphQL schema introspection
  • Microservice architecture mapping
  • API documentation exposure

🔐 Authentication and Authorization

burpsuite --target=target

  • Login mechanism analysis
  • Session management testing
  • OAuth and SSO enumeration
  • JWT token analysis

💻 Client-Side Analysis

js-beautify app.min.js

  • JavaScript code analysis
  • API endpoint extraction
  • Sensitive data in source code
  • Frontend framework vulnerabilities

📱 Mobile and IoT Enumeration

mitmproxy -s extract_endpoints.py

  • Mobile app API analysis
  • IoT device communication
  • Certificate pinning bypass
  • Backend service discovery
