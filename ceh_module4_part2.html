üèóÔ∏è LDAP & DNS Enumeration

LDAP (Lightweight Directory Access Protocol)

LDAP operates on port 389 (636 for LDAPS) and is the foundation of Active Directory. It stores organizational data in a hierarchical structure.

LDAP Enumeration Commands:

ldapsearch -x -h <IP> -s base
ldapsearch -x -h <IP> -b "DC=domain,DC=com"
enum4linux -a <IP>
nmap --script ldap-search <IP>

DNS Enumeration

DNS enumeration reveals domain structure, subdomains, and mail servers. It's often the first step in Active Directory reconnaissance.

DNS Enumeration Techniques:

  • Zone Transfers: Extract complete DNS zone data
  • Reverse DNS: Map IP addresses to hostnames
  • Subdomain Brute Force: Discover hidden subdomains
  • DNS Walking: Enumerate DNS records systematically
dig axfr @<DNS_SERVER> <domain>
dnsrecon -d <domain> -t axfr
fierce -dns <domain>
nslookup -type=any <domain>

üîå RPC & Null Sessions

Remote Procedure Call (RPC) Enumeration

RPC operates on port 135 and enables communication between different processes. It can reveal service information and user accounts.

RPC Enumeration Tools:

rpcclient -U "" -N <IP>
rpcinfo -p <IP>
enum4linux -r <IP>
nmap --script msrpc-enum <IP>

Null Session Attacks

Null sessions exploit the IPC$ share to gain unauthorized access to system information without authentication.

Null Session Commands:

net use \\<IP>\IPC$ "" /u:""
rpcclient -U "" -N <IP>
smbclient //<IP>/IPC$ -N
enum4linux -n <IP>

Information Gathered via Null Sessions:

  • User account information
  • Password policies
  • Share information
  • Registry keys
  • System configuration
